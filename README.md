# sample_app

Python で作ったエンジニア向けの 1 ページポートフォリオサイトです。
問い合わせフォームの送信内容は SQLite に保存されます。

## Overview

このアプリケーションは、自己紹介、スキル、制作実績、問い合わせフォームを 1 つのページにまとめたシンプルなポートフォリオです。
軽量な構成を意識しており、ローカル環境で素早く起動して動作確認できます。

主な用途は次のとおりです。

- エンジニアのプロフィールや実績をわかりやすく掲載する
- 問い合わせフォームから送信された内容を保存する
- Python 製 Web アプリケーションの小さなサンプルとして参照する

## Features

- 1 ページで完結するポートフォリオ表示
- SQLite を使った問い合わせ内容の保存
- `uv` による依存関係管理と実行
- 最小限の構成で読みやすいアプリケーション構造

## Requirements

- Python 3.12+
- uv

## Run

```bash
uv run python -m portfolio_site.app
```

ブラウザで <http://127.0.0.1:8000> を開いてください。

## Database

問い合わせ内容は `data/contacts.db` の `contacts` テーブルに保存されます。
データベースファイルはアプリケーションの実行時に必要に応じて作成されます。
ローカルで動作を確認したい場合は、フォームからテスト送信したあとに SQLite クライアントで保存内容を確認できます。

## Development Notes

このプロジェクトは学習やデモ用途を想定したサンプルアプリです。
本番環境で利用する場合は、入力値の検証、エラーハンドリング、ログ出力、セキュリティ設定、バックアップ方針などを用途に合わせて追加してください。
